# Noise Estimation
Noise estimation in fMRI signals. An AGH-UST project.

# Building
```
make clean && make
```

# Running

```
make run
```

Or...

```
make run CONFIG=path/to/config.conf
```

# Available options:

```
$ ./ne --help
USAGE: ./ne [OPTIONS] CONFIG_FILE
OPTIONS:
	--help         Display this message.
	--no-gui       Disable GUI images.
```
